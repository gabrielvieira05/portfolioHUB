# Lista de Tarefas

Aplicação web simples para criação, gerenciamento e remoção de tarefas diárias. Permite salvar as tarefas localmente no navegador, mantendo a lista ao fechar ou atualizar a página.

---

## Funcionalidades

- Adicionar tarefas novas com um clique ou ao apertar Enter.
- Marcar tarefas como concluídas.
- Remover tarefas individualmente.
- Armazenamento local para manter a lista persistente.

---

## Tecnologias Utilizadas

- HTML5 para a estrutura da página.
- CSS3 para o estilo responsivo e moderno.
- JavaScript para a lógica de interação e armazenamento local.

---

## Como executar

1. Faça o download dos arquivos ou clone o repositório.
2. Abra o arquivo `index.html` diretamente no seu navegador (Chrome, Firefox, Edge etc.).
3. Comece a gerenciar suas tarefas.
